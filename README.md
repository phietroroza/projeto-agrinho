# Projeto Agrinho

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript.

## Sobre o projeto

O objetivo do blog é mostrar conteúdos sobre:
- agricultura,
- sustentabilidade,
- natureza,
- campo e cidade.

## Funcionalidades

- Tema escuro
- Layout responsivo
- Cards organizados com Flexbox
- Efeitos visuais
- Transições suaves

## Tecnologias usadas

- HTML
- CSS
- JavaScript
